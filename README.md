# AntiAntiAFK
Simple Anti AntiAFK script made for Minecraft. __Available only for Linux, Windows version is in development__
# How it works
This program presses W, S, A or D for 0.2 seconds in 3-15 minutes in between presses. You can stop it by either pressing Ctrl+C or you ken set variables ``currentTime.hour``, ``currentTime.minute`` and ``currentTime.second`` to desired time.
__NOTE: You need to run this script as root.__
